# Docs Installation

To run a copy of the docs loaclly, you'll need to install `mkdocs-material`

Install.

```bash
pip install mkdocs-material
```

Run.

```bash
mkdocs serve
```

Then open http://localhost:8000/docs/ to see a copy of the docs.
